포인터 변수: 메모리의 주소 값을 저장하기 위한 변수

int main()
{
	int num=9;	//ex)num의 주소->0x12ff76~0x12ff79(int는 4바이트)
	int* p;		//포인터 p의 선언
	p=&num;		//num의 주소 값을 포인터 변수 p에 저장(p의 값은 0x12ff76)
}

포인터의 크기는 항상 4바이트.
*(dereferenceing operator:간접 참조 연산자): 포인터 변수를 이용해서 간접적으로 접근할
					    때 사용되는 연산자

포인터를 사용할 때에는 꼭 초기화를 시키고 사용.
